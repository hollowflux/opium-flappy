# Opium Flappy 🦇

Flappy Bird-style game with the 005s

## Controls

Space-bar/Left-click: Flap / jump

## Notes

Built with Python + Pygame

Make sure you’re using the virtual environment (.venv) before running

## Run

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python game.py
